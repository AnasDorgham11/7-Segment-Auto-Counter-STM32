# 7-Segment-Auto-Counter-STM32
An embedded systems project that automatically increments a single-digit number (0 → 9) on a 7-segment display every 2 seconds.   When the display reaches 9, it resets back to 0 and continues counting.
---

## 📌 Project Overview

This project demonstrates:

- Interfacing a 7-segment display with STM32
- Configuring GPIO pins as outputs
- Using a timer or delay mechanism
- Creating a time-based counter
- Implementing modulo logic (0–9 loop)
